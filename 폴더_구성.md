## 보조 실행 파일

- `run_pipeline.ps1`: dry-run·approve·execute 단일 실행 진입점. 모듈 이름을 폴더에서 찾으므로 버전이 올라가도 수정이 필요 없다. pwsh 전용이며 장시간 실행은 분리해서 띄운다(계획서 27장).
- `run_dryrun_keepawake.ps1`: run_pipeline.ps1 로 위임하는 래퍼
- `run_execute_keepawake.ps1`: run_pipeline.ps1 로 위임하는 래퍼
- `status_heartbeat.ps1`: 진행 상태 확인
- `prepare_training_view.py`: YOLO 학습 뷰 생성
- `battery_v3_9_preprocessor/tools/verify_ct_selection.py`: dry-run 없이 CT 선정 로직 검증 (계획서 27.9)

원본 데이터, EDA 원본 CSV, 생성 데이터셋과 작업 폴더는 저장소에 포함하지 않는다.
